# telegram-bookshelf-bot
Goodreads API based telegram bot 

## Use it!
You need Goodreads account!

@BooksLoverBot

## Start
- Create your own bot with @BotFather 
- Get Goodreads API developer keys
- Set .environment values
- docker-compose up
